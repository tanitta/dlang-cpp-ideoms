##Contribution
forkした後，masterとは別にbranchを切ってPRを飛ばしてください．

新しくイディオムを追加する際は，Dlangとc++で対応するイディオムをREADME.mdの目次のIdiomsに追加して，詳細を説明したイディオム名のmarkdownをideomsディレクトリに追加してください．正直イディオムというかDlangとC++で対応する文法や機能でも大丈夫かも？
